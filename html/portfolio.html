Memory

Here is a digital version of the card game memory made with jQuery, CSS, and HTML. You have a few seconds to memorize cards' position. Once the words disappear, find the matches. When you've found all the matches your completion time will show up. This was originally for a psychology experiment to measure the effect of the color of the environment on the participant's performance in a memory-based task. I've modified it to be more game-like for this website. Play on a desktop. Above is a gif of the game demo.
